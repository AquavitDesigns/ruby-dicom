is_a_tag?()

Check if a given string appears to be a valid tag (GGGG,EEEE) by regexp matching. The method tests that the string is exactly composed of 4 HEX characters, followed by a comma, then 4 new HEX characters, which constitutes the tag format used by Ruby DICOM.

[Source]

35:     result = true if self =~ /\A[a-fA-F\d]{4},[a-fA-F\d]{4}\z/
36:     return result
